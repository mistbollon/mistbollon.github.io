Mysql数据库

Mysql 基础

一、数据库

1.数据库的概念：

数据库是一种特殊的文件，存储着需要的数据，数据库分为关系型数据库和非关系型数据库，常见的关系型数据库有 orcale、mysql、mysql server、sqlite

2.数据库的安装：

Mysql 数据库安装启动

二、操作指令

1.数据库操作指令

  • 创建数据库:create databases 数据库名

  • 删除数据库:drop database 数据库名

  • 登录:mysql -u用户名 -p密码

  • 退出: quiet; exit;

  • 查看所有数据库:show databases;

  • 查看 mysql 版本：select version();

  • 查看时间:select show();

  • 使用数据库:use 数据库名;

  • 查看当使用数据库：select database();

  • 查看当前数据库的所有数据表:show tables;

2.数据表操作指令
  • 数据表创建，在 use 数据库后再创建

create table 数据表的名字(字段 类型 约束,字段 类型 约束);

约束

类型

例：创建一个 varchar 类型的表

  • 当前使用数据库的字段:desc 库名
  • 查询所有字段:select * from 表名

修改：

  • 添加字段：alter table 表名 add 列名 类型;

    alter table student add birthday datetime;

  • 修改字段：alter table 表名 change 原字段名 新字段名 类型及约束

    alter table student change birth birthday datetime not null;

  • 删除字段:alter table 表名 drop 列名;

    alter table student drop birthday;

  • 删除表: drop 表名

对字段的增删改查

  • 添加：insert into 表名 vlaues(...)

  • 修改：update 表名 set 列1=值,列2=值2 ... where 条件

    update students set name=’李四’,age = 18 where id = 9;

  • 删除：delete from 表名 where 条件

    delete from students wherer id>6;

  • 查询：select id,name,age from student where id>2 and isdelete=0;

三、数据库备份

  • 备份数据库：使用 mysqldump 命令，mysqldump -u用户名 -p密码 数据库名>导出文件名.sql
  • 恢复: 连接 mysql 创建新的数据库，退出连接后使用myslq -u用户名 -p密码 新的数据库名<导入的文件名.sql
